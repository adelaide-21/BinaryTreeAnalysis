# Анализ бинарных деревьев

Реализация алгоритмов для работы с бинарными деревьями на Java.

## Функциональность

1. Подсчет количества листьев - countLeaves()
2. Нахождение глубины дерева - getDepth()
3. Проверка является ли дерево BST - isBST()
4. Сумма значений на каждом уровне - levelSum()
5. Проверка симметрии дерева - isSymmetric()
6. Наименьший общий предок (LCA) - findLCA()
7. Размеры поддеревьев - getSubtreeSizes()

## Компиляция и запуск

`bash
javac BinaryTreeAnalysis.java
java BinaryTreeAnalysis
